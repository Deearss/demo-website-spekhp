# Tech Stack

## Framework
- **Next.js** (App Router) — fullstack React framework
- **TypeScript** — strict mode

## Styling
- **Tailwind CSS v4+** — utility-first, v4 native syntax
- **Inter** — primary font (Google Fonts via next/font)
- **Custom utilities** in `globals.css` (flexc, flexcc, transall, glass, gradient-text)

## Animation

## Icons

- **react-icons 5.6.0**
